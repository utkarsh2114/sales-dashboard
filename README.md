# Sales Dashboard

## Overview

This project analyzes sales performance using Microsoft Excel.

The dashboard was created using:

* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards

## Key Metrics

* Total Sales
* Total Orders
* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Top Selling Products

## Tools Used

* Microsoft Excel
